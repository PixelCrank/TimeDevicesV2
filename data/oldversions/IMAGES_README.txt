# Images manifest for People & Stories

Place images into the project at the following relative paths. You can use any format (`.jpg` recommended).
For each item, add a hero image and a smaller thumbnail (crop-friendly).

**Folder structure (create if missing):**
- `images/people/`
- `images/stories/`

**Recommended sizes**
- Hero: 1600×1200 (or larger), JPG
- Thumb: 400×300, JPG (use center-crop)

See `image_manifest_people_stories.csv` for the checklist.
